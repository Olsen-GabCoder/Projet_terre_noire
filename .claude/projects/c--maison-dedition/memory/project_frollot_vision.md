Le projet initialement appelé "Terre Noire Éditions" est en réalité **Frollot**, une plateforme sociale du livre.

**Why:** L'utilisateur a révélé la vraie vision le 2026-04-01. L'application actuelle (e-commerce maison d'édition) est la base sur laquelle construire la plateforme multi-acteurs.

**How to apply:** 
- Le document de référence est `FROLLOT_VISION.md` à la racine du projet
- Toute nouvelle feature doit s'inscrire dans la roadmap par phases (fondations multi-rôles → marketplace → social → services pro → bibliothèques)
- Le renommage "Terre Noire" → "Frollot" sera fait en Phase 1
- Le stack reste Django + React, pas de migration
- Les 9 apps Django existantes et 17 modèles sont la base à étendre (pas remplacer)
